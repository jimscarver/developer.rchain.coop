Introduction to RChain+-
RChain is a decentralized, economically sustainable public compute infrastructure. While the RChain design is inspired by that of earlier blockchains, it also realizes decades of research across the fields of concurrent and distributed computation, mathematics, and programming language design.

Rholang+-
Rholang is a new language for writing smart contracts that run on the RChain platform.
The language is a reflective higher-order process calculus, based on Milner's asynchronous polyadic pi calculus. As such, it is inherently concurrent, which makes it better suited to distributed programming than languages based on Turing machines or Church's lambda calculus, which are inherently sequential.

What is LADL?+-
LADL is a process to create spatial-behavioral type systems. Such a system not only describes the structure of datatypes, but also the behavior of the processes.
